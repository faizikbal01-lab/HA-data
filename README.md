# HA-data
Healthcare Analytics: A multi-level data analysis project predicting and analyzing hospital readmission rates for diabetic patients. Features statistical modeling, patient segmentation, and clinical insight generation across 50+ variables to improve patient outcomes and operational efficiency.
This project performs a deep-dive analysis into hospital readmission rates using a dataset representing 10 years (1999-2008) of clinical care at 130 US hospitals. The primary goal is to identify factors that lead to patient readmission within 30 days, providing actionable insights for healthcare providers to improve post-discharge care and reduce costs.ðŸ“Š
